# CUR Wizard Integration

This directory contains scripts and templates adapted from the [aws-cur-wizard](https://github.com/Twing-Data/aws-cur-wizard) project, extended to support both AWS and GCP cost analysis dashboards.

**Supported Cloud Providers:**
- AWS Cost and Usage Reports (CUR)
- GCP Billing Export (BigQuery)
